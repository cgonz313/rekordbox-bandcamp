Using the app
1
Set your music folder
Add the folder(s) where your music files live, then click Index Library. The first scan reads metadata from every file — it may take a few minutes. Future scans are instant thanks to a local cache.
2
Connect Bandcamp
A browser window opens. Log in normally — your credentials are never stored. Once logged in, your playlists load here automatically. You have 3 minutes before the session times out; if it does, just click Connect again.
3
Select playlists
Check the playlists you want to export. Use Select all to grab everything at once.
4
Export
Each track is fuzzy-matched against your local files using embedded metadata. The result is a .xml file saved to your chosen export folder.
Importing into rekordbox
1
Open rekordbox
2
Go to Preferences → Advanced → rekordbox xml → Imported Library and point it to your .xml file
3
Your playlists appear in the left sidebar under rekordbox xml
4
Drag them into your main library

On macOS, make sure your music drive is mounted before importing — rekordbox needs to resolve the file paths at import time.

Troubleshooting
Low match rate
Tracks without embedded metadata tags fall back to filename parsing, which is less reliable. Adding proper ID3/AIFF tags with a tool like MusicBrainz Picard will improve results.
Tracks show as missing in rekordbox
The XML uses absolute file paths. If tracks show missing, make sure the music drive is connected and re-import the XML.
"Directory not found" error
Make sure the drive is mounted and the path is correct before indexing.
Browser doesn't open
Re-click Connect Bandcamp. If it still doesn't open, try stopping and restarting the server.
